# 介绍一下代码

**代码里面也会有比较详细的注释**

+ `geometry.cpp geometry.h `向量与矩阵的定义与运算
+ `model.cpp model.h` 模型与纹理的加载
+ `our_gl.cpp our_gl.h`一些图形学算法
+ `test_g.cpp` 用于测试 `geometry.cpp geometry.h` 是否正确



## 运行



```shell
g++ ./tgaimage.cpp ./geometry.cpp ./model.cpp ./our_gl.cpp ./main.cpp -o main
```
